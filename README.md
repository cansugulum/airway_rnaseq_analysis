# Airway RNA-seq Analysis

This repository contains a clean and reproducible RNA-seq analysis workflow using the **airway** dataset.  
All analyses were performed in **R** using standard bioinformatics tools.

## Analysis Overview
- Variance Stabilizing Transformation (VST)
- Principal Component Analysis (PCA)
- Differential Expression Analysis (DESeq2)
- Volcano plot visualisation
- Heatmap of significant genes

## Files
- **airway_rnaseq_analysis.Rmd** — Full analysis pipeline in R Markdown  
- **airway_rnaseq_analysis.html** — Rendered output with visualisations

## Tools & Packages
- R / RStudio  
- DESeq2  
- tidyverse  
- ComplexHeatmap
